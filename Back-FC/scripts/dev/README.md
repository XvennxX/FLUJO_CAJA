# Development Scripts

Scripts para desarrollo y debugging.

## Archivos

- `debug_*.py` - Scripts de debugging de componentes específicos
- `diagnose_*.py` - Scripts de diagnóstico de problemas
- `diagnostico_*.py` - Scripts de diagnóstico en español
- `start_*.bat` - Scripts de inicio de servicios (Windows)
- `check_*.bat` - Scripts de verificación rápida (Windows)

## Uso

```bash
# Debug de autenticación
python scripts/dev/debug_auth.py

# Diagnóstico de transacciones
python scripts/dev/diagnose_transacciones.py
```
