# Tools

Herramientas de verificación y diagnóstico del sistema.

## Archivos

- `check_*.py` - Scripts de verificación de componentes
- `verify_*.py` - Scripts de validación de endpoints y funcionalidad
- `verificar_*.py` - Scripts de verificación de datos

## Uso

```bash
# Verificar esquema de base de datos
python tools/check_schema.py

# Verificar usuarios
python tools/check_users.py

# Verificar endpoints
python tools/verify_endpoints.py
```
